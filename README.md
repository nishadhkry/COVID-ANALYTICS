#Project Overview

This project implements an end-to-end COVID-19 Data Analytics Pipeline using AWS cloud services and modern data engineering practices. The solution follows the Medallion Architecture (Bronze → Silver → Gold) to ingest, transform, and analyze COVID-19 cases and testing data.

The pipeline leverages AWS S3, AWS Glue, PySpark, Athena, and Parquet storage formats to build a scalable analytics platform for COVID-19 reporting and insights.
